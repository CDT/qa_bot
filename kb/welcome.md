# Welcome

This is a sample knowledge base file. Add more files through the sidebar, or drop markdown files directly into the `kb/` directory (restart the server to pick up external changes).

## How it works

- The assistant sees a list of all files (path + title).
- It can grep across files and read any file in full.
- It answers in the language of your question and cites the files it used.

## Limits

- Each file is capped at 10,000 Chinese characters.
- No RAG, no embeddings — just whole-file reads.
